Sistema de Biblioteca e Conversão de Temperaturas


1. Sistema de Biblioteca

Arquivo: `biblioteca.py`

Este programa simula um sistema simples de empréstimo e devolução de livros em uma biblioteca.

Funcionalidades:
- Catálogo inicial com 8 livros predefinidos.
- Identificação de usuários por CPF e nome.
- Empréstimo e devolução de livros.
- Histórico de ações (empréstimos e devoluções) por usuário.
- Dados persistentes salvos em um arquivo JSON.


Como executar:
1. Certifique-se de que você tem Python 3 instalado.
2. Execute o arquivo no terminal:

CMD
python biblioteca.py





2. Conversão de Temperaturas
Arquivo: conversao_temperaturas.py

Este script realiza a conversão de uma lista fixa de temperaturas em Celsius para Fahrenheit, depois converte esses valores para Kelvin e calcula a média das temperaturas em Kelvin.

Como executar:
Execute o script com:

python conversao_temperaturas.py
O programa exibirá os valores convertidos e a média calculada.

Requisitos
Python 3.7 ou superior


Observações
Os programas foram desenvolvidos com foco em clareza e funcionalidade, utilizando apenas recursos da biblioteca padrão do Python. Foram incluídas validações básicas e organização em funções, conforme o escopo de cada exercício.
